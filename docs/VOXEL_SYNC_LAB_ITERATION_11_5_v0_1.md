# Voxel Sync Lab — Iteration 11.5 v0.1

`Iteration11.5/voxel_sync_lab.html` is the named public demo entrypoint for the Iteration 11.5 synchronized voxel lab.

The demo locks five views to one canonical ID spine:

1. Voxel Grouping View
2. Cube Matrix
3. Micro ARC-Room / Stonewalker mesh intake lab
4. Entity set
5. Entity Neural Synth matrix

The package is local-first and runs by opening the HTML file directly. It is a prototype UI foundation, not a production simulation backend.
