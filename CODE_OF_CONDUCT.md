# Code of Conduct

## Our standard

This project aims to keep collaboration direct, useful, and respectful.

Please:

- be constructive
- criticize ideas clearly without personal attacks
- stay focused on improving the project
- keep reports and feedback actionable

## Unacceptable behavior

- harassment
- abusive language
- bad-faith disruption
- spam or repeated low-effort noise

## Enforcement

Maintainers may remove comments, issues, pull requests, or discussions that damage the project environment.
