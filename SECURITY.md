# Security Policy

## Reporting a vulnerability

If you find a security issue, please do not open a public issue with exploit details.

Instead, contact the maintainer privately first and include:

- a clear summary
- affected files or systems
- reproduction steps
- impact estimate
- any suggested fix

## Scope

This is currently an experimental engine repository. Security issues are still taken seriously, especially if they affect file loading, runtime execution, or user data handling.
